Data Units Conversion - List

Here is a list of all known data units conversion:

  • 1 bit = 0.125 bytes
  • 1 byte = 8 bits
  • 1 kilobyte (KB) = 1024 bytes
  • 1 megabyte (MB) = 1024 kilobytes
  • 1 gigabyte (GB) = 1024 megabytes

To convert between data units, you can use the following formulas:

  • Bits to Bytes: divide the number of bits by 8
  • Bytes to Bits: multiply the number of bytes by 8
  • Bytes to Kilobytes: divide the number of bytes by 1024
  • Kilobytes to Bytes: multiply the number of kilobytes by 1024
  • Kilobytes to Megabytes: divide the number of kilobytes by 1024
  • Megabytes to Kilobytes: multiply the number of megabytes by 1024
  • Megabytes to Gigabytes: divide the number of megabytes by 1024
  • Gigabytes to Megabytes: multiply the number of gigabytes by 1024
